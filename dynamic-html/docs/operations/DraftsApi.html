This is the API

createDrafts

Create one or more drafts on the server. These drafts will be automatically synchronized to other clients via `drafts` events. `POST {{ api_url }}/v1/drafts`

URL

https://example.zulipchat.com/api/v1/drafts

HTTP Method

POST

Response Type

JsonSuccess

Parameters

  • Query: drafts List(Draft)

    A JSON-encoded list of containing new draft objects.

deleteDraft

Delete a single draft from the server. The deletion will be automatically synchronized to other clients via a `drafts` event. `DELETE {{ api_url }}/v1/drafts/{draft_id}`

URL

https://example.zulipchat.com/api/v1/drafts/{draft_id}

HTTP Method

DELETE

Response Type

JsonSuccess

Parameters

  • Path: draft_id Integer

    The ID of the draft you want to delete.

editDraft

Edit a draft on the server. The edit will be automatically synchronized to other clients via `drafts` events. `PATCH {{ api_url }}/v1/drafts/{draft_id}`

URL

https://example.zulipchat.com/api/v1/drafts/{draft_id}

HTTP Method

PATCH

Response Type

JsonSuccess

Parameters

  • Path: draft_id Integer

    The ID of the draft to be edited.

  • Query: draft Draft

    A JSON-encoded object containing a replacement draft object for this ID.

getDrafts

Fetch all drafts for the current user. `GET {{ api_url }}/v1/drafts`

URL

https://example.zulipchat.com/api/v1/drafts

HTTP Method

GET

Response Type

JsonSuccess
